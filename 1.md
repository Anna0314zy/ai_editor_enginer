# 🚀 Step 13：SnapEngine 完整实现（AI Prompt）

## 🎯 目标

实现一个**设计工具级吸附系统（Snap Engine）**，支持：

* 元素 ↔ 元素 对齐（left / center / right / top / middle / bottom）
* 元素 ↔ 画布 对齐（水平/垂直居中）
* 输出对齐辅助线（guidelines）
* 阈值吸附（5px）
* 高性能（避免 O(n²)）

---

## 🤖 Prompt（直接复制给 AI）

```text
请实现一个完整的 SnapEngine（TypeScript），用于可视化编辑器中的对齐与吸附系统。

========================
一、功能要求
========================

实现方法：

getSnappedPosition(
  movingElement: Element,
  allElements: Element[],
  nextPosition: { x: number; y: number },
  canvas: { width: number; height: number }
): SnapResult

返回：

type SnapResult = {
  x: number
  y: number
  guides: GuideLine[]
}

type GuideLine = {
  type: 'vertical' | 'horizontal'
  position: number
}

========================
二、对齐规则（必须全部实现）
========================

1️⃣ 元素对元素（X轴）

对齐点：
- left
- center（x + width / 2）
- right（x + width）

规则：
moving.left    ↔ target.left / center / right
moving.center  ↔ target.left / center / right
moving.right   ↔ target.left / center / right

2️⃣ 元素对元素（Y轴）

对齐点：
- top
- middle（y + height / 2）
- bottom（y + height）

规则同上

3️⃣ 元素对画布

X轴：
- center = canvas.width / 2

Y轴：
- middle = canvas.height / 2

========================
三、吸附规则
========================

1. 使用 threshold = 5px
2. 当 |a - b| <= threshold 时触发吸附
3. 优先吸附最近的点（最小差值）
4. X / Y 轴分别计算

========================
四、GuideLine 生成
```
